Nature's Protector
==================
"Nature's protector" sera un site qui enregistre des rapports sur les endroits pollués (principalement en Tunisie).
Les utilisateurs auront la possibilité de se connecter avec leur compte Facebook ou Twitter afin de signaler les emplacement pollués sur une carte.
Le site permettra ainsi l'organisation de campagnes de nettoyage, de relayer l'information sur l'état de notre adorable pays et de sensibiliser les gens sur l'environnement.
